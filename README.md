# deploy-bi-linux

## Prerequest
- wsl
- before running this script need to connect wsl via SSH or open wsl terminal

## Installation
```bash
git clone https://github.com/yuvarajsf/deploy-bi-linux.git
```

```bash
cd deploy-bi-linux
```

## For windows
```bash
py -m pip install -r requirements.txt
```

```bash
py app.py
```
